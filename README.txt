Dawgs Marketplace
=======================================================================

A buy/sell marketplace exclusively for UW Bothell students. Sign in with
your @uw.edu Google account to browse listings, post items for sale, and
connect with other students on campus.

Live site: https://dawgsmarketplace.vercel.app

=======================================================================

HOW TO USE
----------
1. Go to https://dawgsmarketplace.vercel.app
2. Click "Sign in with Google" using your @uw.edu account
3. Browse listings on the home page or use the search bar to find something specific
4. Click "Create Listing" to post an item for sale
5. Go to "My Listings" to view, mark as sold, or delete your posts

=======================================================================

WHAT IT DOES
------------
Dawgs Marketplace is a campus-only platform where UW Bothell students can
list items they want to sell and find items they want to buy. Only students
with a valid @uw.edu Google account can access the site, which keeps the
community trusted and safe.

Implemented features:
- Sign in with Google, restricted to @uw.edu accounts only
- Browse all active listings on the home page
- Search listings by keyword
- Create a listing with a title, description, price, image URL, and contact info
- Mark your own listing as sold
- Delete your own listings
- View all your listings on a personal "My Listings" page

Features not included in final version:
- Edit listing - we went with delete and repost instead
- Email contact button - replaced with a contact info field where sellers can share how they want to be reached
- Category tags, sort by price, and report listing were cut due to time constraints

=======================================================================

TECH STACK
----------
Frontend : React 19 + Vite, deployed on Vercel
Backend  : Spring Boot (Java 21), deployed on Railway
Database : PostgreSQL via Supabase
Auth     : Google OAuth 2.0 + JWT
